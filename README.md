# Контрольная работа №3

Реализовано PWA-приложение (менеджер задач) с поддержкой офлайн-режима, WebSockets и Push-уведомлений.

## Реализованный функционал

- **App Shell & Service Worker:** кэширование статики, работа в офлайне.
- **Web App Manifest:** возможность установки приложения на устройство.
- **HTTPS:** локальный сервер на Node.js с самоподписанными сертификатами.
- **WebSockets:** синхронизация задач в реальном времени (`Socket.io`).
- **Push-уведомления:** системные уведомления для новых задач и запланированных напоминаний (с возможностью отложить на 5 минут).

## Запуск проекта

1. Установить зависимости:
   ```bash
   npm install
   ```

2. Сгенерировать SSL-сертификаты. Выполнить в папке certs команды:
   ```bash
   mkcert -install
   mkcert localhost 127.0.0.1 ::1
   ```

3. Запустить сервер:
   ```bash
   npm start
   ```

4. Открыть приложение в браузере по адресу:
   **https://localhost:3001**
